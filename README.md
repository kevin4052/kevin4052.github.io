# kevinhernandez.dev

Personal portfolio site for Kevin Hernandez — a full-stack software developer based in Austin, TX.

**Live:** [kevinhernandez.dev](https://kevinhernandez.dev)

## Stack

- Single-page HTML/CSS (no build step)
- Google Fonts (Inter)
- Hosted on GitHub Pages with a custom domain

## Structure

```
index.html      — main portfolio page
robots.txt      — search-engine crawling directives
sitemap.xml     — XML sitemap for indexing
CNAME           — custom domain config
resume.md       — markdown version of resume
assets/         — CSS, fonts, images, JS
```

## SEO

- Canonical URL, Open Graph, and Twitter Card meta tags
- JSON-LD `Person` structured data
- Sitemap and robots.txt for crawlers
